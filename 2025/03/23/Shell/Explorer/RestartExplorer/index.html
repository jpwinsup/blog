タスク マネージャーからエクスプローラーを再起動する方法

Last Update: 2025-03-23
Shell►Explorer
feedback 共有

本記事はマイクロソフト社員によって公開されております。

こんにちは、Windows サポート チームの栗木です。
エクスプローラーは、Windows のユーザー インターフェースを管理する重要なプロセスであり、デスクトップ、タスク バー、ファイル エクスプローラーなどを担当しています。Windows を使用していると、何らかの要因でエクスプローラーが突然応答しなくなったり、ファイルやフォルダが正しく表示されなくなることがあります。こうした問題が発生した場合、システム全体を再起動するのではなく、エクスプローラーだけを再起動することで問題を解決できることがあります。
本記事では、タスク マネージャーを使用してエクスプローラーを再起動する方法について説明します。

適用対象の OS

  • すべての Windows OS

※ Windows 10 や Windows Server 2022 以前の OS を使用している環境では、タスク マネージャーの UI がスクリーン ショットと異なりますが、操作方法に違いはありません。

再起動手順

1. タスクマネージャーを起動

タスク マネージャーを起動するには、キーボードの Ctrl + Shift + Esc キーを同時に押すか、タスク バーを右クリックして [タスク マネージャー] を選択します。

2. [ユーザー] 画面に切り替え

タスク マネージャーが開いたら、サイド メニューから [ユーザー] 画面に切り替えます。この画面では、現在ログインしているユーザーごとにプロセスが表示されます。Windows 10 や Windows Server 2022 以前の OS を使用している環境では、上部に配置されている [ユーザー] タブを選択してください。

3. エクスプローラー右クリック

再起動したいエクスプローラーを実行しているユーザー（例：user01）のプロセス一覧から [エクスプローラー] を探します。見つけたら、そのプロセスを右クリックします。

4. コンテキストメニューから [再起動] をクリック

右クリックすると表示されるコンテキストメニューから [再起動] を選択します。これにより、エクスプローラーが再起動され、デスクトップやタスク バーがリフレッシュされます。環境によっては、エクスプローラーが複数ありますが、その場合はそれぞれのエクスプローラーを再起動してください。

補足情報

再起動されたエクスプローラーを実行しているユーザーが、切断状態の場合には、エクスプローラーは終了した後にすぐに起動せず、ユーザーが接続状態になったときに起動します。

他のユーザーのエクスプローラーを再起動したい場合には、管理者権限を持つユーザーでタスク マネージャーを起動して作業する必要があります。管理者権限をもたないユーザー（例：user01）でログインし、タスク マネージャーを実行すると、以下のスクリーン ショットのように他のユーザーのプロセス一覧が表示できないようになっています。
