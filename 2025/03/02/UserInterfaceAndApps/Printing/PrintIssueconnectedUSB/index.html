USB 接続のプリンターで意図しない文字列が印刷される

Last Update: 2025-03-02
User Interface and Apps►Printing (印刷)
feedback 共有

こんにちは、Windows サポート チームの神田です。
Windows の更新プログラムを適用すると、印刷処理に一部想定しない動作が発生する事が確認されており、こちらのブログにて確認している事象および対応策について、お知らせいたします。
※ 3/26 修正プログラムの提供開始について更新しました。

事象

USB で接続したプリンターで印刷を実行した際に、以下のような文字列が印刷されます。

要因

Windows 10 および 11 において、2025 年 1 月第 4 週におよび 2 月第 2 週以降に提供された更新プログラムの一部に、印刷処理に想定しない動作を行うものが含まれており上記に記載した問題が発生します。

発生が確認されているバージョンは、以下の通りです。

Windows 10 22H2 にて以下およびこれより新しい更新プログラムをインストールしている環境

  • KB5050081 (2025 年 1 月第 4 週)
    2025 年 1 月 28 日 — KB5050081 (OS ビルド 19045.5440) プレビュー - Microsoft サポート
  • KB5051974 (2025 年 2 月第 2 週)
    2025 年 2 月 11 日 — KB5051974 (OS ビルド 19044.5487 および 19045.5487) - Microsoft サポート

Windows 11 23H2 にて以下およびこれより新しい更新プログラムをインストールしている環境

  • KB5050092 (2025 年 1 月第 4 週)
    2025 年 1 月 29 日 — KB5050092 (OS ビルド 22621.4830 および 22631.4830) プレビュー - Microsoft サポート
  • KB5051989 (2025 年 2 月第 2 週)
    2025 年 2 月 11 日 — KB5051989 (OS ビルド 22621.4890 および 22631.4890) - Microsoft サポート

また、本事象について、以下の「既知の問題と通知」 Web ページにも記載されています。

Windows 10、バージョン 22H2 の既知の問題と通知 | Microsoft Learn
Windows 10, version 22H2 known issues and notifications | Microsoft Learn (英語)
Windows 11、バージョン 22H2 の既知の問題と通知 | Microsoft Learn
Windows 11, version 22H2 known issues and notifications | Microsoft Learn (英語)
Windows 11, version 23H2 known issues and notifications | Microsoft Learn (英語)

本事象については更新プログラムが提供されています。下記から更新プログラムをインストールして事象に改善が見られるか、ご確認いただきますようお願いいたします。

Windows 10 22H2 向け更新プログラム
March 25, 2025—KB5053643 (OS Build 19045.5679) Preview
Windows 11 22H2, 23H2 向け更新プログラム
March 25, 2025—KB5053657 (OS Builds 22621.5126 and 22631.5126) Preview

何からの事情で更新プログラムのインストールが行えない場合は、下記の回避策についてご検討ください。

回避策 #1

USB 接続でプリンターをご利用いただいている場合に発生する事象となります。
ネットワーク プリンターでは発生しないため、ネットワーク機能を有するプリンターをご利用の場合は、ネットワーク プリンターの設定を行っていただければ本事象は発生いたしません。

回避策 #2

KIR (Known Issue Rollback) という仕組みで問題のあるコードを無効にする設定を展開できます。
KIR のためのグループポリシーをインストールするモジュールは、前述の各 OS の「既知の問題と通知」のページの「グループ ポリシーをグループ ポリシー名でダウンロードします」の項目よりダウンロード可能です。

KIR の適用、展開方法の詳細については、下記の技術情報をご参照ください。
グループ ポリシーを使用して既知の問題のロールバックを展開する方法

※ 補足
Windows 11 24H2 については、以下の更新プログラムにて修正していますが、一部回避できていない状況を確認しており調査中となります。
2025 年 1 月 28 日 — KB5050094 (OS ビルド 26100.3037) プレビュー - Microsoft サポート
2025 年 2 月 11 日 — KB5051987 (OS ビルド 26100.3194) - Microsoft サポート

弊社製品の想定しない動作により、ご迷惑をおかけしており誠に申し訳ございません。

よろしくお願いいたします。
