Last Update: 2022-09-02

feedback е…ұжңү

жң¬иЁҳдәӢгҒҜгҖҒгғһгӮӨгӮҜгғӯгӮҪгғ•гғҲзӨҫе“ЎгҒ«гӮҲгҒЈгҒҰе…¬й–ӢгҒ•гӮҢгҒҰгҒҠгӮҠгҒҫгҒҷгҖӮ

гҒ“гӮ“гҒ«гҒЎгҒҜгҖҒWindows гӮөгғқгғјгғҲ гғҒгғјгғ гҒ§гҒҷгҖӮ
д»ҠеӣһгҒҜ Storage Migration Service (SMS) гҒЁ Azure File Sync (AFS) гҒ®йҖЈжҗәж©ҹиғҪгҒ«гҒҠгҒ‘гӮӢеҲ¶йҷҗдәӢй …гҒ«гҒӨгҒ„гҒҰиЁҳијүгҒ•гҒӣгҒҰгҒ„гҒҹгҒ гҒҚгҒҫгҒҷгҖӮ

жҰӮиҰҒ

Storage Migration Service гҒЁ Azure File Sync гҒ®йҖЈжҗәж©ҹиғҪгӮ’еҲ©з”ЁгҒ—гҒҹе ҙеҗҲгҖҒStorage Migration Service гҒ®и»ўйҖҒгҒҢзөӮгӮҸгӮүгҒӘгҒ„гҖӮ

и©ізҙ°

Storage Migration Service гҒҜз§»иЎҢе…ғгӮөгғјгғҗгҒӢгӮүз§»иЎҢе…ҲгҒ®гӮөгғјгғҗгҒёгғ•гӮЎгӮӨгғ«гӮ’и»ўйҖҒгҒ—гҒҫгҒҷгҖӮгҒ•гӮүгҒ«гҖҒз§»иЎҢе…ҲгҒ®гӮөгғјгғҗгҒӢгӮүгҖҒAzure File Sync гҒЁйҖЈжҗәгҒ—гҒҰгҖҒAzure дёҠгҒ«гғ•гӮЎгӮӨгғ«гӮ’дҝқеӯҳгҒҷгӮӢгҒ“гҒЁгӮӮеҸҜиғҪгҒ§гҒҷгҖӮгҒ—гҒӢгҒ—гҒӘгҒҢгӮүгҖҒд»ҘдёӢгҒ®жқЎд»¶гҒ«еҗҲиҮҙгҒ—гҒҹе ҙеҗҲгҖҒз§»иЎҢе…ғгӮөгғјгғҗгҒӢгӮүз§»иЎҢе…ҲгҒ®гӮөгғјгғҗгҒёгғ•гӮЎгӮӨгғ«гҒҢи»ўйҖҒгҒ•гӮҢгҒҫгҒӣгӮ“гҖӮ

<жқЎд»¶>

гғ» и»ўйҖҒгҒ•гӮҢгӮӢгғ•гӮЎгӮӨгғ«гҒ®з·ҸйҮҸгҒҢгҖҒStorage Migration Service гҒ® вҖңгғңгғӘгғҘгғјгғ гҒ®жңҖе°ҸгҒ®з©әгҒҚе®№йҮҸвҖқ гҒ®еҖӨгӮҲгӮҠеӨ§гҒҚгҒ„
гғ» Azure File Sync еҒҙгҒ®иЁӯе®ҡгҒ§гҒӮгӮӢ вҖңгғңгғӘгғҘгғјгғ гҒ®з©әгҒҚй ҳеҹҹгғқгғӘгӮ·гӮ’жҢҮе®ҡгҒ—гҒҹеүІеҗҲгҒ®з©әгҒҚгӮ№гғҡгғјгӮ№гҒ«еёёжҷӮж®ӢгҒҷвҖқ гҒ®еҖӨгӮҲгӮҠгӮӮ
     Storage Migration Service еҒҙгҒ® вҖңгғңгғӘгғҘгғјгғ гҒ®жңҖе°ҸгҒ®з©әгҒҚе®№йҮҸвҖқ гҒ®еҖӨгҒҢеӨ§гҒҚгҒ„е ҙеҗҲ

// гғңгғӘгғҘгғјгғ гҒ®жңҖе°ҸгҒ®з©әгҒҚе®№йҮҸгҒ®з”»йқў (Storage Migration Service)


// гғңгғӘгғҘгғјгғ гҒ®з©әгҒҚй ҳеҹҹгғқгғӘгӮ·гӮ’жҢҮе®ҡгҒ—гҒҹеүІеҗҲгҒ®з©әгҒҚгӮ№гғҡгғјгӮ№гҒ«еёёжҷӮж®ӢгҒҷ (Azure File Sync)


дҫӢгҒҲгҒ°гҖҒд»ҘдёӢгҒ®гӮҲгҒҶгҒӘз’°еўғгӮ’жғіе®ҡгҒ—гҒҫгҒҷгҖӮ

з’°еўғдҫӢ)
и»ўйҖҒе…ҲгҒ®гғңгғӘгғҘгғјгғ : E гғүгғ©гӮӨгғ–
E гғүгғ©гӮӨгғ–гҒ®гғңгғӘгғҘгғјгғ гӮөгӮӨгӮә: 100 GB
гғңгғӘгғҘгғјгғ гҒ®жңҖе°ҸгҒ®з©әгҒҚе®№йҮҸ : 50 %
и»ўйҖҒгҒҷгӮӢгғ•гӮЎгӮӨгғ« : 20 GB гҒ®гғ•гӮЎгӮӨгғ«гӮ’ 3 еҖӢ
гғңгғӘгғҘгғјгғ гҒ®з©әгҒҚй ҳеҹҹгғқгғӘгӮ·гӮ’жҢҮе®ҡгҒ—гҒҹеүІеҗҲгҒ®з©әгҒҚгӮ№гғҡгғјгӮ№гҒ«еёёжҷӮж®ӢгҒҷ : 20 %

гҒ“гҒ®е ҙеҗҲгҖҒгғңгғӘгғҘгғјгғ гҒ®жңҖе°ҸгҒ®з©әгҒҚе®№йҮҸгҒҜ 100 GB гҒ® 50% гҒЁгҒӘгӮҠгҒҫгҒҷгҒ®гҒ§гҖҒ50 GBгҒ«гҒӘгӮҠгҒҫгҒҷгҖӮгҒ“гҒ®зҠ¶жіҒгҒ§гҖҒStorage Migration Service гҒҢи»ўйҖҒгӮ’й–Ӣе§ӢгҒ—гҒҹе ҙеҗҲгҒ«гҒҜгҖҒ2 еҖӢзӣ®гҒ®гғ•гӮЎгӮӨгғ«гҒҫгҒ§гҒҜи»ўйҖҒгҒ•гӮҢгҒҫгҒҷгҒҢгҖҒ3 еҖӢзӣ®гҒ®гғ•гӮЎгӮӨгғ«гҒҜи»ўйҖҒгҒ•гӮҢгҒҫгҒӣгӮ“гҖӮгҒ“гӮҢгҒҜгҖҒжңҖе°ҸгҒ®з©әгҒҚе®№йҮҸгҒ® 50 GB гӮ’и¶…гҒҲгҒҰгҒ—гҒҫгҒҶгҒҹгӮҒгҒ§гҒҷгҖӮStorage Migration Service гҒҜ 3 еҖӢзӣ®гҒ®гғ•гӮЎгӮӨгғ«гҒҢи»ўйҖҒгҒ§гҒҚгӮӢз©әгҒҚе®№йҮҸгҒҢзўәдҝқгҒ•гӮҢгӮӢгҒҫгҒ§и»ўйҖҒгӮ’дёҖжҷӮзҡ„гҒ«еҒңжӯўгҒ®зҠ¶ж…ӢгҒ®гҒҫгҒҫгҒ«гҒӘгӮҠгҖҒи»ўйҖҒеҮҰзҗҶгҒҢзөӮгӮҸгӮүгҒӘгҒ„зҠ¶ж…ӢгҒЁгҒӘгӮҠгҒҫгҒҷгҖӮ

гҒҫгҒҹгҖҒвҖқгғңгғӘгғҘгғјгғ гҒ®з©әгҒҚй ҳеҹҹгғқгғӘгӮ·гӮ’жҢҮе®ҡгҒ—гҒҹеүІеҗҲгҒ®з©әгҒҚгӮ№гғҡгғјгӮ№гҒ«еёёжҷӮж®ӢгҒҷвҖқгҒҢ 20 % гҒ«гҒӘгҒЈгҒҰгҒ„гӮӢгҒҹгӮҒгҖҒ80 GB д»ҘдёҠгҒ«гҒӘгӮүгҒӘгҒ„гҒЁгҖҒAzure File Sync гҒ®йҡҺеұӨеҢ–еҮҰзҗҶгҒҢеӢ•гҒҚгҒҫгҒӣгӮ“гҖӮгҒқгҒ®гҒҹгӮҒгҖҒгғңгғӘгғҘгғјгғ гҒ«з©әгҒҚе®№йҮҸгӮӮзўәдҝқгҒ•гӮҢгҒӘгҒ„зҠ¶ж…ӢгҒЁгҒӘгӮҠгҒҫгҒҷгҖӮ

еӣһйҒҝзӯ–

гҒ“гҒ®дәӢиұЎгӮ’еӣһйҒҝгҒҷгӮӢгҒҹгӮҒгҒ«д»ҘдёӢгҒ®гҒ„гҒҡгӮҢгҒӢгӮ’е®ҹж–ҪгҒҷгӮӢгҒ“гҒЁгӮ’гҒ”жӨңиЁҺгҒ„гҒҹгҒ гҒ‘гӮҢгҒ°гҒЁеӯҳгҒҳгҒҫгҒҷгҖӮ

// еӣһйҒҝзӯ–1

и»ўйҖҒгҒҷгӮӢгғ•гӮЎгӮӨгғ«гҒ®з·ҸйҮҸгҒҢ вҖңгғңгғӘгғҘгғјгғ гҒ®жңҖе°ҸгҒ®з©әгҒҚе®№йҮҸвҖқ гӮ’и¶…гҒҲгҒӘгҒ„гӮҲгҒҶгҒ«гҖҒдәӢеүҚгҒ«з©әгҒҚе®№йҮҸгӮ’зўәдҝқгҒҷгӮӢгҖӮгӮӮгҒ—гҒҸгҒҜ вҖңгғңгғӘгғҘгғјгғ гҒ®жңҖе°ҸгҒ®з©әгҒҚе®№йҮҸвҖқ гӮ’йқһеёёгҒ«дҪҺгҒ„ж•°еҖӨгҒ«еӨүжӣҙгҒҷгӮӢгҖӮ

// еӣһйҒҝзӯ–2

Azure File Sync гҒЁйҖЈжҗәгҒ—гҒҰеҲ©з”ЁгҒҷгӮӢе ҙеҗҲгҒ«гҒҜгҖҒд»ҘдёӢгҒ®иЁҲз®—ејҸгӮ’жәҖгҒҹгҒҷгӮҲгҒҶгҒ«гҖҒAzure File Sync еҒҙгҒ®иЁӯе®ҡгҒЁ Storage Migration Service гҒ®иЁӯе®ҡгӮ’иЎҢгҒҶгҖӮ

<иЁҲз®—ејҸ>
(Azure File Sync еҒҙгғӯгғјгӮ«гғ«гғңгғӘгғҘгғјгғ гӮөгӮӨгӮә Г— гғңгғӘгғҘгғјгғ гҒ®з©әгҒҚй ҳеҹҹгғқгғӘгӮ·гӮ’жҢҮе®ҡгҒ—гҒҹеүІеҗҲгҒ®з©әгҒҚгӮ№гғҡгғјгӮ№гҒ«еёёжҷӮж®ӢгҒҷ(%))вү§((гғңгғӘгғҘгғјгғ е®№йҮҸ Г— гғңгғӘгғҘгғјгғ гҒ®жңҖе°ҸгҒ®з©әгҒҚе®№йҮҸ(%)) + жңҖеӨ§гғ•гӮЎгӮӨгғ«гӮөгӮӨгӮә)

вҖ» Azure File Sync еҒҙгҒ®гғӯгғјгӮ«гғ«гғңгғӘгғҘгғјгғ гӮөгӮӨгӮәгҒҜгӮӘгғігғ—гғ¬гғҹгӮ№еҒҙгҒ®гғңгғӘгғҘгғјгғ гӮ’жҢҮгҒ—гҒҰгҒҠгӮҠгҒҫгҒҷгҖӮ

гҒ„гҒӢгҒҢгҒ§гҒ—гҒҹгҒ§гҒ—гӮҮгҒҶгҒӢгҖӮжң¬жҠ•зЁҝгҒҢе°‘гҒ—гҒ§гӮӮзҡҶж§ҳгҒ®гҒҠеҪ№гҒ«з«ӢгҒҰгҒ°е№ёгҒ„гҒ§гҒҷгҖӮ
жң¬жғ…е ұгҒ®еҶ…е®№пјҲж·»д»ҳж–ҮжӣёгҖҒгғӘгғігӮҜе…ҲгҒӘгҒ©гӮ’еҗ«гӮҖпјүгҒҜгҖҒдҪңжҲҗж—ҘжҷӮгҒ§гҒ®гӮӮгҒ®гҒ§гҒӮгӮҠгҖҒдәҲе‘ҠгҒӘгҒҸеӨүжӣҙгҒ•гӮҢгӮӢе ҙеҗҲгҒҢгҒӮгӮҠгҒҫгҒҷгҖӮ

ж¬ЎгҒ®иЁҳдәӢ
ж„ҸеӣігҒ—гҒӘгҒ„гӮ·гғЈгғғгғҲгғҖгӮҰгғігғ»еҶҚиө·еӢ•гҒҢзҷәз”ҹгҒ—гҒҹгӮү
