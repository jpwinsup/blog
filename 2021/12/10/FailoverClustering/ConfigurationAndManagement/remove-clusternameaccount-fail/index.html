PowerShell を使用したクラスターのドメイン間マイグレーションが失敗する不具合について

Last Update: 2021-12-10
Failover Clustering►Configuration and Management
feedback 共有

本記事はマイクロソフト社員によって公開されております。

いつも弊社製品をご利用いただきまして誠にありがとうございます。
日本マイクロソフトの Windows サポートチームです。

クラスターのドメイン移行について、弊社から以下情報が公開されており、Windows Server 2019 では
PowerShell コマンドを使用した新たな方法が記載されております。

Title : Failover Cluster domain migration
URL : https://docs.microsoft.com/en-us/windows-server/failover-clustering/cluster-domain-migration
項目 : Windows Server 2019、Migrating a cluster to a new domain をご参照ください。

2024 年 9 月以前に公開情報に記載の PowerShell コマンドを用いた方法では、ドメイン間マイグレーションが正常に完了しないことが確認されておりました。
具体的には、”Remove-ClusterNameAccount” にて後述しますエラーが発生し、一連のマイグレーション手順が失敗する事象が報告されておりました。

現時点では、公開情報に記載の手順が更新されており、以下のいずれかを指定して Remove-ClusterNameAccount を実行することで
事象が解消されることを確認しております。

  • -Cluster オプションを削除する
    例） Remove-ClusterNameAccount -DeleteComputerObjects
  • -Cluster <Node Name> を指定する
    例） Remove-ClusterNameAccount -Cluster Node1 -DeleteComputerObjects

Remove-ClusterNameAccount 実行時のエラー

2024 年 9 月以前に公開情報に記載の移行手順では、”Remove-ClusterNameAccount” コマンドにて、
以下のエラーが発生し正常に動作しないことが確認できておりました。

PS C:\ > Remove-ClusterNameAccount -Cluster < CNO > -DeleteComputerObjects Remove-ClusterNameAccount :
クラスターへのアクセスに使用された名前は、現在、使用できなくなっています。今すぐ接続するには、クラスター ノード名またはクラスターによって公開されている別の任意の名前を試してみてください。
発生場所 行:1 文字:1
Remove-ClusterNameAccount -Cluster < CNO > -DeleteComputerObjec … -~
   CategoryInfo : ResourceBusy: (:) [Remove-ClusterNameAccount], ClusterCmdletException
   FullyQualifiedErrorId : ClusterStatusNotReady,Microsoft.FailoverClusters.PowerShell.RemoveClusterNameAccount

PS C:\ > Remove-ClusterNameAccount -Cluster < VCO > -DeleteComputerObjects Remove-ClusterNameAccount :
クラスター名のスペルを確認してください。スペルが正しい場合は、ネットワークに問題がある可能性があります。クラスター ノードがオンになっており、ネットワークに接続されていることを確認してください。それ以外の問題の場合は、ネットワーク管理者に問い合わせてください。
    RPC サーバーを利用できません。
発生場所 行:1 文字:1
Remove-ClusterNameAccount -Cluster < VCO > -DeleteComputerObjects
-~
   CategoryInfo : 接続エラー: (: ) [Remove-ClusterNameAccount]、ClusterCmdletException
   FullyQualifiedErrorId : ClusterRpcConnection,Microsoft.FailoverClusters.PowerShell.RemoveClusterNameAccount

推奨対策事項

公開情報に記載のクラスターのドメイン移行方法を更新しております。
以下の公開情報に記載の手順で Note 欄に記載の内容を確認の上、移行作業を実施くださいますようお願いいたします。

Title : Failover Cluster domain migration
URL : https://docs.microsoft.com/en-us/windows-server/failover-clustering/cluster-domain-migration
項目 : Migrating a cluster to a new domain
