#1 Windows Commercial 紹介 Blog (組織概要&サポートの仕事編)

本記事はマイクロソフト社員によって公開されております。

私たちのチームでは、一緒にサポート エンジニア部門 (CSS: Customer Service & Support) を盛り上げてくれるエンジニアの方を募集しております。本記事では、私たち Windows Commercial の組織やサポートの仕事と働き方の概要についてご案内します。

目次：

  1. CSS & Windows Commercial の組織概要
  2. Windows サポートの仕事とは？

1. CSS & Windows Commercial の組織概要

サポート エンジニア部門 (CSS: Customer Service & Support)

CSS サポート部門は Azure、Microsoft 365、Windows など、製品に対応した専任の組織があり、各製品への技術サポートを提供しております。サポート エンジニアは、高度な技術力 (複雑な問題解決) とお客様に寄り添ったコミュニケーションを持って、お客様の問題解決に努めています。

Windows Commercial

CSS の一組織である Windows Commercial は更に 6 つの技術エリア (チーム) に細分化され、合計 100 名を超えるエンジニアが在籍しております。Windows サポートでは、有償サポート契約を締結している企業様からの Windows OS に関連する技術サポートを提供しております。

※ Windows Commercial には技術エリアに関係なく緊急案件を対応する特別チーム CRT (Critical Response Team) も所属しています。

2. Windows サポートの仕事とは？

主な業務：お問い合わせ対応

Windows サポートの主な業務としては、Windows OS 製品に関したお客様からの技術的なお問い合わせの対応を通じて、お客様が抱える課題解決をサポートすることになります。ここでいう「お客様」とは、Microsoft が提供しているサポート サービスである Microsoft Unified や Premier Support for Partners 契約を締結しているお客様となりますので、通常 Microsoft 製品をご利用いただいている企業様や他企業様向けに IT 支援サービスを提供されている SIer 様などが主なお客様となります。

<お問い合わせ対応の流れ>

お客様からのお問い合わせ対応の流れを紹介します。お問い合わせを Windows サポートにて受付すると、基本的には以下のサイクルを回すことで、課題解決を行います。ただし、発生事象の種類やお問い合わせの内容により特定ステップが省略されることや、追加の情報採取などのステップが加わることがありますので、あくまで一例として紹介させていただきます。

Microsoft の Windows サポートの業務は、お客様が抱えられている問題を迅速かつ効果的に解決することを目指し、オーナーシップを持って最後まで対応します。また、複数の製品に関わる事象については、活発なコラボレーションを通じて最適な解決策を見つけ、お客様に「問い合わせてよかった」と思っていただけるサポートサービスを提供します。

問い合わせ対応以外の業務：

お客様からのお問い合わせ対応以外にも、ナレッジの展開や、業務改善のプロジェクト活動など、様々な仕事があり本人のやる気次第でグローバルとの協業含め活躍できるチャンスがあります。

3. 働き方について（Work Flexibility）

Flex 制度

Windows サポートにおける働き方は非常に柔軟です。サポートエンジニアの業務時間は、サポートの窓口の時間に合わせて基本的には 9:00-17:30 ですが、Flex 制度を採用しており、ご家庭の事情に合わせて一時的に抜けることも可能です。また、その際のバックアップなどもチームでフォローする体制があり、子育て中のお母さんやお父さんエンジニアが多数ご活躍されております。

リモートワーク

オフィス出社とリモートワークのどちらかを選ぶことができ、その両方を組み合わせたハイブリッド勤務も可能です。さらに、フルリモート勤務のエンジニアも複数名おり、西は九州から北は北海道まで各地からサポート業務に従事されております。

このように、Windows サポートでは多様な働き方が可能であり、エンジニアが最適な環境で働けるように必要な支援を整えてきておりますので、ご興味をお持ちの方は是非、Search Jobs | Microsoft Careers より応募をご検討ください。

また、次回は Windows Commercial に所属する各チームで扱う製品や活かせる、また学べる技術について紹介させていただきます！
