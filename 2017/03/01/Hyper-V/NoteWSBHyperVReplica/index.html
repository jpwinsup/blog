вҖ» жң¬иЁҳдәӢгҒҜејҠзӨҫгҒ® Technet гҒӢгӮүз§»иЎҢгҒ—гҒҹиЁҳдәӢгҒ§гҒҷгҖӮ
гҒ„гҒӨгӮӮејҠзӨҫиЈҪе“ҒгӮ’гҒ”еҲ©з”ЁгҒ„гҒҹгҒ гҒҚгҒҫгҒ—гҒҰиӘ гҒ«гҒӮгӮҠгҒҢгҒЁгҒҶгҒ”гҒ–гҒ„гҒҫгҒҷгҖӮж—Ҙжң¬гғһгӮӨгӮҜгғӯгӮҪгғ•гғҲгҒ® Windows гӮөгғқгғјгғҲгғҒгғјгғ гҒ§гҒҷгҖӮ

д»ҠеӣһгҒҜгҖҒHyper-V гғ¬гғ—гғӘгӮ«гҒЁ Windows Server гғҗгғғгӮҜгӮўгғғгғ—гӮ’дҪөз”ЁгҒҷгӮӢйҡӣгҒ«гҖҒгғ¬гғ—гғӘгӮұгғјгӮ·гғ§гғігҒҢеҒңжӯўгҒҷгӮӢдәӢиұЎгҒ«гҒӨгҒ„гҒҰгҒ”жЎҲеҶ…гҒ•гҒӣгҒҰгҒ„гҒҹгҒ гҒҚгҒҫгҒҷгҖӮ

в– гғ¬гғ—гғӘгӮұгғјгӮ·гғ§гғігҒЁгҒҜпјҹ
гӮігғігғ”гғҘгғјгӮҝгӮ„гӮҪгғ•гғҲгӮҰгӮ§гӮўгҒ®з®ЎзҗҶгҒҷгӮӢгғҮгғјгӮҝгҒЁеҗҢгҒҳеҶ…е®№гҒ®гғ¬гғ—гғӘгӮ«(иӨҮиЈҪ)гӮ’дҪңжҲҗгҒҷгӮӢгҒ“гҒЁгҒ§гҒҷгҖӮ
гғӣгӮ№гғҲгӮөгғјгғҗгҒ®гғҮгғјгӮҝжғ…е ұгҒЁдҪңжҲҗгҒ—гҒҹгғ¬гғ—гғӘгӮ«гҒ®еҶ…е®№гӮ’е®ҡжңҹзҡ„гҒ«еҗҢжңҹгҒҷгӮӢгҒҹгӮҒгҖҒдёҮгҒҢдёҖгғӣгӮ№гғҲгӮөгғјгғҗгҒ«йҡңе®ігҒҢзҷәз”ҹгҒ—гҒҹе ҙеҗҲгҒ§гӮӮгҖҒ
дҪңжҲҗгҒ—гҒҹгғ¬гғ—гғӘгӮ«гҒӢгӮүгӮ·гӮ№гғҶгғ гӮ’зЁјеғҚгҒ—з¶ҡгҒ‘гӮӢгҒ“гҒЁгҒҢгҒ§гҒҚгҒҫгҒҷгҖӮ
гҒқгҒ®гҒҹгӮҒгҖҒгғ¬гғ—гғӘгӮұгғјгӮ·гғ§гғігӮ’иЎҢгҒҶгҒ“гҒЁгҒ§гӮ·гӮ№гғҶгғ гҒ®й«ҳеҸҜз”ЁжҖ§гӮ’й«ҳгӮҒгӮӢгҒ“гҒЁгҒ«гҒӨгҒӘгҒҢгӮҠгҒҫгҒҷгҖӮ

в– Hyper-V гғ¬гғ—гғӘгӮ«гҒЁгҒҜпјҹ
Hyper -V гғ¬гғ—гғӘгӮ«гҒҜгҖҒ Windows Server 2012гҒӢгӮүе®ҹиЈ…гҒ•гӮҢгҒҹ Hyper-V гҒ®ж©ҹиғҪгҒ®дёҖйғЁгҒ§гҒҷгҖӮ
Hyper-V гғӣгӮ№гғҲгҒ§дҪңжҲҗгҒ—гҒҹд»®жғігғһгӮ·гғігҒ§Hyper-Vгғ¬гғ—гғӘгӮ«гҒ®ж©ҹиғҪгӮ’жңүеҠ№гҒ«гҒҷгӮӢгҒЁгҖҒ
Hyper-VгғӣгӮ№гғҲгҒ®д»®жғігғһгӮ·гғігҒ®гғ¬гғ—гғӘгӮ«гҒҢHyper-V гғ¬гғ—гғӘгӮ«гҒ«дҪңжҲҗгҒ•гӮҢгҖҒйҡҸжҷӮгғ¬гғ—гғӘгӮұгғјгӮ·гғ§гғігӮ’иЎҢгҒҶгҒ“гҒЁгҒҢгҒ§гҒҚгҒҫгҒҷгҖӮ

в– Hyper-V гғ¬гғ—гғӘгӮ«гҒЁ Windows Server гғҗгғғгӮҜгӮўгғғгғ—гӮ’дҪөз”ЁгҒҷгӮӢйҡӣгҒ®жіЁж„ҸзӮ№
Windows Server жЁҷжә–гҒ® Windows Server гғҗгғғгӮҜгӮўгғғгғ—гҒ§гҒҜ Hyper-V гғ¬гғ—гғӘгӮ«з’°еўғгҒ®гғҗгғғгӮҜгӮўгғғгғ—гҒ«гҒҜеҜҫеҝңгҒ—гҒҰгҒҠгӮҠгҒҫгҒӣгӮ“гҖӮ

гҒқгҒ®гҒҹгӮҒгҖҒHyper-V гғ¬гғ—гғӘгӮ«гҒ§гғ¬гғ—гғӘгӮұгғјгӮ·гғ§гғігӮ’иЎҢгҒЈгҒҰгҒ„гӮӢйҡӣгҒ«
Windows Server гғҗгғғгӮҜгӮўгғғгғ—гҒ«гҒҰд»®жғігғһгӮ·гғігҒ®гғҗгғғгӮҜгӮўгғғгғ—гӮ’е®ҹиЎҢгҒҷгӮӢгҒЁ
Hyper-V гғ¬гғ—гғӘгӮ«гҒЁHyper-V гғӣгӮ№гғҲгҒ®гғ¬гғ—гғӘгӮұгғјгӮ·гғ§гғігҒҢеҲҮж–ӯгҒ•гӮҢгӮӢдәӢиұЎгҒҢзҷәз”ҹгҒҷгӮӢгҒ“гҒЁгҒҢгҒ”гҒ–гҒ„гҒҫгҒҷгҖӮ

в– еҜҫеҝңзӯ–
Hyper-V гғ¬гғ—гғӘгӮ«гҒ«еҜҫеҝңгҒ—гҒҹгғҗгғғгӮҜгӮўгғғгғ—гӮўгғ—гғӘгӮұгғјгӮ·гғ§гғігҒ®гҒ”еҲ©з”ЁгӮ’гҒ”жӨңиЁҺгҒҸгҒ гҒ•гҒ„гҖӮ
ејҠзӨҫиЈҪе“ҒгҒ§гҒҜгҖҒSystem Center Data Protection Manager (DPM) 2012 R2 д»ҘйҷҚгҒ®гғҗгғјгӮёгғ§гғігҒӢгӮүеҜҫеҝңгҒ—гҒҰгҒҠгӮҠгҒҫгҒҷгҖӮ

гҒҫгҒҹгҖҒд»ҘдёӢгҒ®пј’зӮ№гӮ’иҖғж…®гҒҷгӮӢгҒ“гҒЁгҒ§еӣһйҒҝгҒҷгӮӢгҒ“гҒЁгҒҢгҒ§гҒҚгҒҫгҒҷгҖӮ

  1. гғҗгғғгӮҜгӮўгғғгғ—гҒ®й–Ӣе§ӢеүҚгҒ«гғ¬гғ—гғӘгӮұгғјгӮ·гғ§гғігӮ’дёҖжҷӮеҒңжӯўгҒҷгӮӢгҖӮ
  2. гғҗгғғгӮҜгӮўгғғгғ—е®ҢдәҶеҫҢгҖҒгғ¬гғ—гғӘгӮұгғјгӮ·гғ§гғігӮ’еҶҚй–ӢгҒҷгӮӢгҖӮ

вҖ»гғ¬гғ—гғӘгӮұгғјгӮ·гғ§гғігҒ®еҶҚй–Ӣ / дёҖжҷӮеҒңжӯўгҒҜгҖҒд»ҘдёӢгҒ® PowerShellгӮігғһгғігғүгҒ«гӮҲгӮҠе®ҹиЎҢгҒҷгӮӢгҒ“гҒЁгҒҢгҒ§гҒҚгҒҫгҒҷгҖӮ

гғ¬гғ—гғӘгӮұгғјгӮ·гғ§гғігҒ®еҶҚй–ӢгҖҖгҖҖпјҡResume-VMReplication
гғ¬гғ—гғӘгӮұгғјгӮ·гғ§гғігҒ®дёҖжҷӮеҒңжӯўпјҡSuspend-VMReplication

жң¬жғ…е ұгҒ®еҶ…е®№пјҲж·»д»ҳж–ҮжӣёгҖҒгғӘгғігӮҜе…ҲгҒӘгҒ©гӮ’еҗ«гӮҖпјүгҒҜгҖҒдҪңжҲҗж—ҘжҷӮзӮ№гҒ§гҒ®гӮӮгҒ®гҒ§гҒӮгӮҠгҖҒдәҲе‘ҠгҒӘгҒҸеӨүжӣҙгҒ•гӮҢгӮӢе ҙеҗҲгҒҢгҒӮгӮҠгҒҫгҒҷгҖӮ
