жң¬иЁҳдәӢгҒҜгҖҒгғһгӮӨгӮҜгғӯгӮҪгғ•гғҲзӨҫе“ЎгҒ«гӮҲгҒЈгҒҰе…¬й–ӢгҒ•гӮҢгҒҰгҒҠгӮҠгҒҫгҒҷгҖӮ

гҒ“гӮ“гҒ«гҒЎгҒҜгҖҒWindows гӮөгғқгғјгғҲ гғҒгғјгғ гҒ®иҝ‘и—ӨгҒ§гҒҷгҖӮ
жң¬ж—ҘгҒҜиЎЁйЎҢгҒ®е•ҸйЎҢгҒ«гҒӨгҒ„гҒҰгҒ®жғ…е ұгӮ’гҒ”зҙ№д»ӢгҒ„гҒҹгҒ—гҒҫгҒҷгҖӮ

е•ҸйЎҢгҒ®жҰӮиҰҒ

Microsoft Excel гӮ„д»–зӨҫгӮҪгғ•гғҲгӮҰгӮ§гӮўзӯүгҒ®ж©ҹиғҪгҒЁгҒ—гҒҰгҖҒзӣҙжҺҘ Outlook гғЎгғғгӮ»гғјгӮёгҒ®дҪңжҲҗгӮҰгӮЈгғігғүгӮҰгӮ’й–ӢгҒҸж©ҹиғҪгҒҢе®ҹиЈ…гҒ•гӮҢгҒҰгҒ„гӮӢе ҙеҗҲгҒҢгҒӮгӮҠгҒҫгҒҷгҒҢгҖҒгҒ“гҒ®ж©ҹиғҪгҒ®еҲ©з”ЁжҷӮгҒ«гҖҒгӮўгғ—гғӘгӮұгғјгӮ·гғ§гғігҒҢеҝңзӯ”гҒӘгҒ—гҒ«гҒӘгӮӢе•ҸйЎҢгҒҢе ұе‘ҠгҒ•гӮҢгҒҰгҒҠгӮҠгҒҫгҒҷгҖӮ
гҒҫгҒҹгҒ“гҒ®е•ҸйЎҢгҒҜ вҖңд»ҘеүҚгҒ®гғҗгғјгӮёгғ§гғігҒ® Microsoft IME гӮ’дҪҝгҒҶвҖқ иЁӯе®ҡгӮ’гӮӘгғігҒ«гҒ—гҒҰгҒ„гҒҹгҒ гҒҸгҒ“гҒЁгҒ§еӣһйҒҝгҒҢеҸҜиғҪгҒ§гҒҷгҖӮ

е•ҸйЎҢгҒ®и©ізҙ°

ејҠзӨҫ Excel иЈҪе“ҒгӮ„д»–зӨҫгӮҪгғ•гғҲгӮҰгӮ§гӮўиЈҪе“ҒзӯүгҒ®ж©ҹиғҪгҒЁгҒ—гҒҰгҖҒOutlook гғЎгғғгӮ»гғјгӮёгҒ®дҪңжҲҗгӮҰгӮЈгғігғүгӮҰгӮ’й–ӢгҒҸж©ҹиғҪгҒҢе®ҹиЈ…гҒ•гӮҢгҒҰгҒ„гӮӢе ҙеҗҲгҒҢгҒӮгӮҠгҒҫгҒҷгҖӮ
гҒ“гҒ®гӮҲгҒҶгҒӘж©ҹиғҪгҒҜ MAPISENDMAILW API гҒ«гӮҲгӮҠе®ҹиЈ…еҸҜиғҪгҒ§гҒҷгҒҢгҖҒгӮўгғ—гғӘгӮұгғјгӮ·гғ§гғігҒҢгҒ“гҒЎгӮүгҒ® API гӮ’дҪҝз”ЁгҒ—гҒҰ Outlook гғЎгғғгӮ»гғјгӮёгҒ®дҪңжҲҗгӮҰгӮЈгғігғүгӮҰгӮ’й–ӢгҒҚ [е®ӣе…Ҳ] [CC] гҒӘгҒ©гҒ®гғңгӮҝгғігҒӢгӮүйҖЈзөЎе…ҲгӮҰгӮЈгғігғүгӮҰгӮ’й–ӢгҒҸгҒЁгҖҒе‘јгҒіеҮәгҒ—е…ғгҒ®гӮўгғ—гғӘгӮұгғјгӮ·гғ§гғіеҸҠгҒігҖҒOutlook гӮўгғ—гғӘгӮұгғјгӮ·гғ§гғігҒҢгғҸгғігӮ°гӮўгғғгғ—гҒ—гҖҒеҝңзӯ”гҒ§гҒҚгҒӘгӮӢе•ҸйЎҢгҒҢзҷәз”ҹгҒҷгӮӢе ҙеҗҲгҒҢгҒӮгӮҠгҒҫгҒҷгҖӮ

гҒ“гҒ®е•ҸйЎҢгҒҜ Windows 10 гҒ®ж—Ҙжң¬иӘһе…ҘеҠӣгҖҒдёӯеӣҪиӘһе…ҘеҠӣгҒ«й–ўгҒҷгӮӢе®ҹиЈ…дёҠгҒ®е•ҸйЎҢгҒ«гӮҲгӮҠзҷәз”ҹгҒ—гҒҰгҒ„гӮӢгҒ“гҒЁгҒҢзү№е®ҡгҒ•гӮҢгҒҰгҒҠгӮҠгҖҒгҒҫгҒҹ Windows 11 гҒ«гҒҠгҒ„гҒҰж”№е–„гҒ—гҒҰгҒҠгӮҠгҒҫгҒҷгҖӮгҒҫгҒҹ Windows 10 гҒ«гҒҠгҒ„гҒҰгҒҜ вҖңд»ҘеүҚгҒ®гғҗгғјгӮёгғ§гғігҒ® Microsoft IME гӮ’дҪҝгҒҶвҖқ иЁӯе®ҡгӮ’гӮӘгғігҒ«гҒ—гҒҰгҒ„гҒҹгҒ гҒҸгҒ“гҒЁгҒ§гҖҒеӣһйҒҝгҒ„гҒҹгҒ гҒҸгҒ“гҒЁгҒҢеҸҜиғҪгҒ§гҒҷгҖӮ

е•ҸйЎҢгҒ®зҷәз”ҹжқЎд»¶

гҒ“гҒ®е•ҸйЎҢгҒҜж—Ҙжң¬иӘһе…ҘеҠӣ (гҒҫгҒҹгҒҜдёӯеӣҪиӘһе…ҘеҠӣ) гӮ’еҲ©з”ЁгҒ—гҒҰгҒ„гӮӢ Windows 10 version 20H2 пҪһ 22H2 гҒҫгҒҹгҒҜ Windows 10 LTSC 2021 гҒ«гҒҠгҒ„гҒҰзҷәз”ҹгҒҷгӮӢгҒ“гҒЁгҒҢзўәиӘҚгҒ•гӮҢгҒҰгҒҠгӮҠгҒҫгҒҷгҖӮ

гҒҫгҒҹгҒ“гҒ®е•ҸйЎҢгҒҜ MAPISENDMAILW API гҒ«гҒҠгҒ„гҒҰгғўгғјгғҖгғ« гӮҰгӮЈгғігғүгӮҰгӮ’й–ӢгҒҸ MAPI_DIALOG гғ•гғ©гӮ°гӮ’д»ҳеҠ гҒ—гҒҰ Outlook гғЎгғғгӮ»гғјгӮёгҒ®дҪңжҲҗгӮҰгӮЈгғігғүгӮҰгӮ’й–ӢгҒ„гҒҰгҒ„гӮӢе ҙеҗҲгҒ«гҒ®гҒҝзҷәз”ҹгҒ—гҒҫгҒҷгҒ®гҒ§гҖҒејҠзӨҫ Excel иЈҪе“ҒгҒ®гғҗгғјгӮёгғ§гғігҒ«гӮҲгӮҠгҖҒгҒҫгҒҹд»–зӨҫгӮҪгғ•гғҲгӮҰгӮ§гӮўиЈҪе“ҒгҒ®е®ҹиЈ…гҒ«гӮҲгӮҠгҖҒзҷәз”ҹгҒҷгӮӢе ҙеҗҲгҒЁзҷәз”ҹгҒ—гҒӘгҒ„е ҙеҗҲгҒЁгҒҢгҒӮгӮҠгҒҫгҒҷгҖӮ

е•ҸйЎҢгҒёгҒ®еҜҫеҮҰж–№жі•

д»ҠжҷӮзӮ№гҒ§гҒ“гҒ®е•ҸйЎҢгҒҜ [е®ӣе…Ҳ] [CC] зӯүгҒ®гғңгӮҝгғігҒӢгӮүйҖЈзөЎе…ҲгӮҰгӮЈгғігғүгӮҰгӮ’й–ӢгҒ„гҒҹе ҙеҗҲгҒ®гҒҝгҖҒзҷәз”ҹгҒҢзўәиӘҚгҒ•гӮҢгҒҰгҒ„гҒҫгҒҷгҖӮгҒқгҒ®гҒҹгӮҒгҖҒгҒ“гӮҢгӮүгҒ®гғңгӮҝгғігӮ’жҠјгҒ•гҒӘгҒ„гҒ“гҒЁгҒ§е•ҸйЎҢгҒ®зҷәз”ҹгӮ’еӣһйҒҝгҒҷгӮӢгҒ“гҒЁгҒҢеҸҜиғҪгҒ§гҒҷгҖӮ

гҒҫгҒҹгҒ“гҒ®е•ҸйЎҢгҒ®зҷәз”ҹгҒ«гҒҜгҖҒWindows 10 version 2004 д»ҘйҷҚгҒ§е®ҹиЈ…гҒ•гӮҢгҒҹж–°гҒ—гҒ„ Microsoft IME гҒ®е®ҹиЈ…гҒҢй–ўдҝӮгҒ—гҒҰгҒҠгӮҠгҖҒвҖқд»ҘеүҚгҒ®гғҗгғјгӮёгғ§гғігҒ® Microsoft IME гӮ’дҪҝгҒҶвҖқ иЁӯе®ҡгӮ’гӮӘгғігҒ«гҒ—гҒҰгҒ„гҒҹгҒ гҒҸеҪўгҒ§гӮӮеӣһйҒҝгҒҢеҸҜиғҪгҒ§гҒҷгҖӮиЁӯе®ҡж–№жі•зӯүгҒ«гҒӨгҒ„гҒҰгҒҜгҒ“гҒЎгӮүгҒ®иЁҳдәӢгӮ’гҒ”еҸӮз…§гҒҸгҒ гҒ•гҒ„гҖӮ

  • д»ҘеүҚгҒ®гғҗгғјгӮёгғ§гғігҒ® IME (е…ҘеҠӣж–№ејҸгӮЁгғҮгӮЈгӮҝгғј) гҒ«жҲ»гҒҷ
  • д»ҘеүҚгҒ®гғҗгғјгӮёгғ§гғігҒ® Microsoft IME гҒ®гҒ”дҪҝз”ЁгҒ«гҒӨгҒ„гҒҰ

гҒӘгҒҠж®ӢеҝөгҒӘгҒҢгӮүгҖҒе•ҸйЎҢгҒ®зҷәз”ҹеҫҢгҒ«е…ғгҒ®гӮўгғ—гғӘгӮұгғјгӮ·гғ§гғігҒ®зҠ¶ж…ӢгҖҒе…ғгҒ® Outlook гғЎгғғгӮ»гғјгӮёз”»йқўгҒ«жҲ»гӮӢж–№жі•гҒҜгҒ”гҒ–гҒ„гҒҫгҒӣгӮ“гҖӮдҫӢгҒҲгҒ°гӮҝгӮ№гӮҜ гғһгғҚгғјгӮёгғЈгғјзӯүгҒӢгӮү Outlook гҒҠгӮҲгҒігҖҒе‘јгҒіеҮәгҒ—е…ғгҒ®гӮўгғ—гғӘгӮұгғјгӮ·гғ§гғігӮ’зөӮдәҶгҒ—гҒҰгҖҒеҶҚеәҰгӮўгғ—гғӘгӮұгғјгӮ·гғ§гғігҒ®й–Ӣе§ӢгҒӢгӮүеҶҚй–ӢгҒҷгӮӢгҒӘгҒ©гҒ®гҒ”еҜҫеҮҰгӮ’гҒ„гҒҹгҒ гҒ‘гҒҫгҒҷгӮҲгҒҶгҖҒгҒҠйЎҳгҒ„гҒ„гҒҹгҒ—гҒҫгҒҷгҖӮ

жң¬жғ…е ұгҒҢгҒҠе®ўж§ҳгҒ®е•ҸйЎҢи§ЈжұәгҒ®дёҖеҠ©гҒЁгҒӘгӮҠгҒҫгҒ—гҒҹгӮүе№ёгҒ„гҒ§гҒҷгҖӮ
