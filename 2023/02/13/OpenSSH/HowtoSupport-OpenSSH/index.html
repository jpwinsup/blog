OpenSSH for Windows のサポートについて

Last Update: 2023-02-14
OpenSSH
feedback 共有

本記事はマイクロソフト社員によって公開されております。

こんにちは、Windows サポートチームの國重です。
本記事では、最近お問い合わせいただくことの多い OpenSSH for Windows のサポートについてご紹介いたします。

Windows 10、Windows Server 2019 に OpenSSH for Windows をインストールされている方や、インストールを検討されている方もいらっしゃるかと思いますので、今後の運用にあたって本記事を参考していただけますと幸いでございます。

OpenSSH for Windows のサポートについて

OpenSSH for Windows はオープン ソースとして GitHub 上に公開しており、オープン ソースのコミュニティ ベースで対応をさせていただいております。
各ユーザー様から問題の報告、修正案の提案などのフィードバックを受け付けておりますので、OpenSSH for Windows 特有の問題が発生した場合や、改善案などのフィードバックがございましたら、Issues (問題の報告)、Pull Requests (修正案) にてご報告いただけますと幸いでございます。

なお、Windows 10 Version 1809, Windows Server 2019 以降の OS では Features On Demand 経由で OpenSSH のインストールが可能となりますが、GitHub でインストールした OpenSSH と比較してサポートしている内容に違いはございません。
Features On Demand 経由でインストールされる OpenSSH は GitHub に公開している最新版よりも古いバージョンになるため、バージョンの違いによる未修正の問題などを考慮される場合は、GitHub から最新版の OpenSSH をインストールしていただけますと幸いでございます。

Premier/Unified サポートでの取り扱いについて

OpenSSH for Windows はオープン ソースのコミュニティ ベースで対応しておりますが、弊社の Premier/Unified サポートをご契約いただいているお客様につきましては、同サポートにて OpenSSH for Windows の設定、調査のご支援をさせていただいております。

　<ご支援している内容>
　・OpenSSH for Windows の設定方法
　・OpenSSH for Windows をお使いいただく中で発生したトラブルの回避方法の調査

ただし、Github からダウンロードしたソース コードをコンパイル、ビルドする方法や、ソース コードの改修方法、またソース コードの改修作業といったご支援は承ることができません。
また、OpenSSH for Windows の問題であることが判明し、かつ、GitHub 上に公開している最新版のバージョンでも修正されていない場合は、弊社サポートで解析した調査結果をもとにお客様にて GitHub の Issues (問題の報告) に報告していただく必要がございますので、予め、ご理解いただけますと幸いでございます。
