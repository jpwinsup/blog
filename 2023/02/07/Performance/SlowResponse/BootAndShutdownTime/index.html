起動時刻とシャットダウン時刻

Last Update: 2023-02-07
Performance►Slow Response
feedback 共有

本記事はマイクロソフト社員によって公開されております。

こんにちは、Windows サポート チームの栗木です。
本記事では、システムのイベントログから OS の起動時刻やシャットダウン時刻を確認する方法をご紹介いたします。なお、OS の起動・シャットダウン時刻になりますので、実際に端末の電源ボタンを入れた時刻やシャットダウンが完了し、電源が落ちた時刻とは誤差があるのに加えて、正確には OS の起動時刻を示すイベントログが記録される前やシャットダウン時刻を示すイベントログが記録される後にも OS の処理はございます。

OS の起動・シャットダウン時刻を確認することにより、起動遅延やシャットダウン遅延の問題個所を絞り込める可能性がございます。例えば、端末の電源を入れた時刻と OS 起動時刻を比べて時間があいている場合は、その間の処理で時間がかかったと判断できたり、シャットダウン開始時刻とシャットダウン時刻を比べて時間が空いている場合は、シャットダウン処理のいずれかの処理で時間がかかっていると判断できます。

適用対象の OS

  • すべての Windows OS

起動時刻 (ID 6005)

OS の起動時刻は、イベントログ サービスの開始を示すイベントログ（レベル：情報、ソース：EventLog、イベント ID：6005）から確認ができます。

シャットダウン開始時刻 (ID 1074)

OS のシャットダウン開始時刻は、どのプロセスによりシャットダウン処理が開始されたかを示すイベントログ（レベル：情報、ソース：User32、イベント ID：1074）から確認ができます。

シャットダウン時刻 (ID 13)

OS のシャットダウン時刻は、シャットダウン時刻を示すイベントログ（レベル：情報、ソース：Kernel-General、イベント ID：13）から確認ができます。
