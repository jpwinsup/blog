вҖ» жң¬иЁҳдәӢгҒҜејҠзӨҫгҒ® Technet гҒӢгӮүз§»иЎҢгҒ—гҒҹиЁҳдәӢгҒ§гҒҷгҖӮ
гҒ„гҒӨгӮӮејҠзӨҫиЈҪе“ҒгӮ’гҒ”еҲ©з”ЁгҒ„гҒҹгҒ гҒҚгҒҫгҒ—гҒҰиӘ гҒ«гҒӮгӮҠгҒҢгҒЁгҒҶгҒ”гҒ–гҒ„гҒҫгҒҷгҖӮж—Ҙжң¬гғһгӮӨгӮҜгғӯгӮҪгғ•гғҲгҒ® Windows гӮөгғқгғјгғҲгғҒгғјгғ гҒ§гҒҷгҖӮ

д»ҠвҪҮгҒҜгғңгғӘгғҘгғјгғ гӮ’гғһгӮҰгғігғҲ/гғҮгӮЈгӮ№гғһгӮҰгғігғҲгҒҷгӮӢгҒЁгҒҚгҒӘгҒ©гҒ«дҪҝвҪӨгҒҷгӮӢ Mountvol гӮігғһгғігғүгҒ®гҖҒ/D гӮӘгғ—гӮ·гғ§гғігҒЁ /P гӮӘгғ—гӮ·гғ§гғігҒ®е·®з•°гҒ«гҒӨгҒ„гҒҰгҖҒгҒҠи©ұгҒ•гҒӣгҒҰгҒ„гҒҹгҒ гҒҚгҒҫгҒҷгҖӮ

Monuntvol гӮігғһгғігғүгҒ§гғңгғӘгғҘгғјгғ  гғһгӮҰгғігғҲ гғқгӮӨгғігғҲгӮ’еүҠйҷӨгҒ—гҒҹгӮҠгҖҒгғүгғ©гӮӨгғ– гғ¬гӮҝгғјгӮ’еүҠйҷӨгҒҷгӮӢгӮӘгғ—гӮ·гғ§гғігҒЁгҒ—гҒҰгҖҒ/D гӮӘгғ—гӮ·гғ§гғігҒЁ /P гӮӘгғ—гӮ·гғ§гғігҒҢгҒӮгӮҠгҒҫгҒҷгҖӮ

гҒ©гҒЎгӮүгҒ®гӮӘгғ—гӮ·гғ§гғігҒ§гӮӮеүҠйҷӨгҒ—гҒҹгғүгғ©гӮӨгғ– гғ¬гӮҝгғјгӮ„гғһгӮҰгғігғҲ гғқгӮӨгғігғҲгӮ’вҪӨгҒ„гҒҹгӮўгӮҜгӮ»гӮ№гҒҜгҒ§гҒҚгҒӘгҒҸгҒӘгӮӢзӮ№гҒ«йҒ•гҒ„гҒҜгҒӮгӮҠгҒҫгҒӣгӮ“гҖӮ

гҒҹгҒ гҒ—гҖҒгҒ“гӮҢгӮүдёЎгӮӘгғ—гӮ·гғ§гғігҒ®йҮҚиҰҒгҒӘе·®з•°гҒҢгҒӮгӮҠгҒҫгҒҷгҖӮ

гҒқгҒ®е·®з•°гҒҜгҖҒ/D гӮӘгғ—гӮ·гғ§гғігҒ§гҒҜжҢҮе®ҡгҒ•гӮҢгҒҹгғһгӮҰгғігғҲ гғқгӮӨгғігғҲгҒҢеүҠйҷӨгҒ•гӮҢгӮӢгҒ®гҒҝгҒ§гҖҒOS гҒ®еҶ…йғЁзҡ„гҒ«гҒҜгғһгӮҰгғігғҲзҠ¶ж…ӢгҒ®гҒҫгҒҫгҒЁгҒӘгӮӢгҒ®гҒ«еҜҫгҒ—гҖҒ/P гӮӘгғ—гӮ·гғ§гғігҒ«гҒҰгғһгӮҰгғігғҲгғқгӮӨгғігғҲгӮ’еүҠйҷӨгҒ—гҒҹе ҙеҗҲгҒ«гҒҜе®Ңе…ЁгҒ«гғҮгӮЈгӮ№гғһгӮҰгғігғҲгҒ•гӮҢгҒҹзҠ¶ж…ӢгҒЁгҒӘгӮӢйғЁеҲҶгҒ«гҒӘгӮҠгҒҫгҒҷгҖӮ

йҖҡеёёгҒӮгҒҫгӮҠгҒ“гҒ®гӮӘгғ—гӮ·гғ§гғігҒ®е·®з•°гҒҢе•ҸйЎҢгҒ«гҒӘгӮӢгҒ“гҒЁгҒҜгҒӮгӮҠгҒҫгҒӣгӮ“гҒҢгҖҒOS гӮ’д»ӢгҒ•гҒҡгҒ«гӮ№гғҲгғ¬гғјгӮёгҒ«гӮўгӮҜгӮ»гӮ№гӮ’вҫҸгҒҶгӮ·гӮ№гғҶгғ зӯүгӮ’гҒ”еҲ©вҪӨгҒ®е ҙеҗҲгҒ«гҒҜгҖҒдёҠиЁҳгғһгӮҰгғігғҲзҠ¶ж…ӢгҒ®е·®з•°гӮ’ж„ҸиӯҳгҒ—гҒҰгҒҠгҒҸеҝ…иҰҒгҒҢгҒӮгӮҠгҒҫгҒҷгҖӮ

вҫјж©ҹиғҪгҒӘгӮ№гғҲгғ¬гғјгӮё гӮ·гӮ№гғҶгғ гҒ§гҒҜ OS гӮ’д»ӢгҒ•гҒҡгҒ«гғҮгӮЈгӮ№гӮҜиӨҮиЈҪзӯүгӮ’вҫҸгҒҲгӮӢгӮӮгҒ®гҒҢгҒӮгӮҠгҖҒгҒ“гҒ®зЁ®гҒ®гғҮгӮЈгӮ№гӮҜж“ҚдҪңгҒҢ OS гҒӢгӮүгғһгӮҰгғігғҲгҒ•гӮҢгҒҰеҲ©вҪӨгҒ•гӮҢгҒҰгҒ„гӮӢзҠ¶ж…ӢгҒ§вҫҸгӮҸгӮҢгӮӢгҒЁгҖҒOS гҒҢз®ЎзҗҶгҒ—гҒҰгҒ„гӮӢжғ…е ұгҒЁе®ҹйҡӣгҒ®гғҮгӮЈгӮ№гӮҜдёҠгҒ®гғҮгғјгӮҝгҒ«дёҚвјҖиҮҙгҒҢзҷәвҪЈгҒ—гҖҒгғҮгӮЈгӮ№гӮҜдёҠгҒ®гғ•гӮЎгӮӨгғ«гӮ·гӮ№гғҶгғ гҒ®дёҚж•ҙеҗҲгӮ’жӢӣгҒҸгҒ“гҒЁгҒҢгҒӮгӮҠгҒҫгҒҷгҖӮ

гғ•гӮЎгӮӨгғ« гӮ·гӮ№гғҶгғ гҒ®дёҚж•ҙеҗҲгҒҜгғҮгӮЈгӮ№гӮҜдёҠгҒ®гғҮгғјгӮҝгҒ®йҡңе®ігҒ«гӮӮгҒӨгҒӘгҒҢгӮӢеҸҜиғҪжҖ§гҒҢгҒӮгӮҠгҒҫгҒҷгҒ®гҒ§гҖҒгҒ“гҒ®гӮҲгҒҶгҒӘгӮ·гӮ№гғҶгғ гӮ’гҒ”еҲ©вҪӨгҒ®гҒҠе®ўж§ҳгҒ§гҒҜгҒ”жіЁж„ҸгҒҸгҒ гҒ•гҒ„гҖӮ

гҒӘгҒҠгҖҒдёЎиҖ…гҒҢгҒ©гҒЎгӮүгҒ®зҠ¶ж…ӢгҒ«гҒӮгӮӢгҒӢгҒҜ Mountvol гӮігғһгғігғүгӮ’гӮӘгғ—гӮ·гғ§гғігҒӘгҒ—гҒ§е®ҹвҫҸгҒ—гҒҹзөҗжһңгҒӢгӮүеҲӨж–ӯгҒ§гҒҚгҒҫгҒҷгҖӮ

гӮ·гғЈгғүгӮҰ гӮігғ”гғјгҒ®иЁӯе®ҡгӮҝгғ–гҒҜгӮЁгӮҜгӮ№гғ—гғӯгғјгғ©гҒӢгӮүгғүгғ©гӮӨгғ–гҒ®гғ—гғӯгғ‘гғҶгӮЈгӮ’йҒёжҠһгҒҷгӮӢгҒЁиЎЁвҪ°гҒ•гӮҢгҒҫгҒҷгҖӮ

гҒ“гҒ®иЁӯе®ҡгӮҝгғ–гҒ§гҒҜгӮ·гӮ№гғҶгғ гҒ«еӯҳеңЁгҒ—гҒҰгҒ„гӮӢгғңгғӘгғҘгғјгғ гҒҢвјҖиҰ§иЎЁвҪ°гҒ•гӮҢгҒҫгҒҷгҒ®гҒ§гҖҒеҗ„ Mountvol гӮігғһгғігғүе®ҹвҫҸеҫҢгҒ®иЎЁвҪ°гӮ’д»ҘдёӢгҒ«иӘ¬жҳҺгҒ—гҒҫгҒҷгҖӮ

дҫӢ 3 : еӣі 1 гҒ®зҠ¶ж…ӢгҒӢгӮү Mountvol E: /P гӮ’вҫҸгҒЈгҒҹзҠ¶ж…Ӣ

E: гҒҜгғңгғӘгғҘгғјгғ гҒЁгҒ—гҒҰгӮўгғігғһгӮҰгғігғҲгҒ•гӮҢгҒҹзҠ¶ж…ӢгҒ«гҒӘгҒЈгҒҰгҒ„гҒҫгҒҷгҒ®гҒ§гҖҒвјҖиҰ§гҒ«гғӘгӮ№гғҲгҒ•гӮҢгҒҰгҒ„гҒҫгҒӣгӮ“гҖӮ

дёҠиЁҳзҠ¶ж…ӢгҒӢгӮүгҖҒгғңгғӘгғҘгғјгғ гҒ®зҠ¶ж…ӢгҒ®е·®з•°гҒҢгҒ”зўәиӘҚгҒ„гҒҹгҒ гҒ‘гҒҹгҒЁжҖқгҒ„гҒҫгҒҷгҖӮ
