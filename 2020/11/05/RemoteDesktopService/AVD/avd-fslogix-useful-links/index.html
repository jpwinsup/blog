AVD 新機能

MSIX App Attach

アプリケーションを OS と分離し、アプリケーションをユーザーログオン時に OS にアタッチ（取り付ける）ことができる機能です。

MSIX アプリのアタッチとは
https://docs.microsoft.com/ja-jp/azure/virtual-desktop/what-is-app-attach

Microsoft Learn - Azure Virtual Desktop 用の MSIX アプリ アタッチを使用してアプリケーションを展開する
https://docs.microsoft.com/ja-jp/learn/modules/m365-wvd-application-management/

Smart Card Authentication using KDC Proxy on AVD

KDC プロキシ サービスを利用して、スマートカードでの認証をできるようにする機能です。

Kerberos キー配布センター プロキシを構成する
https://docs.microsoft.com/ja-jp/azure/virtual-desktop/key-distribution-center-proxy

Azure Monitor for Azure Virtual Desktop

Azure Virtual Desktop 環境を監視する機能です。

Azure Virtual Desktop 向けの Azure Monitor を使用してデプロイを監視する
https://docs.microsoft.com/ja-jp/azure/virtual-desktop/azure-monitor

AVD での Teams のメディア最適化

Azure Virtual Desktop 上の Microsoft Teams は、チャットとコラボレーションをサポートしています。
メディア最適化が加わると、さらに通話と会議の機能もサポートされます。

Azure Virtual Desktop で Microsoft Teams を使用する
https://docs.microsoft.com/ja-jp/azure/virtual-desktop/teams-on-avd

スケーリング ツール

ピーク時以外の使用時間帯にセッション ホスト VM をシャットダウンして割り当て解除し、ピーク時間帯に再びオンにして再割り当てする機能です。

Azure Automation を使用してセッション ホストをスケーリングする
https://docs.microsoft.com/ja-jp/azure/virtual-desktop/set-up-scaling-script

AVD SSO with AD FS

AD FS を利用し、AVD に SSO でのサインインができる機能です。

公式アナウンス
https://techcommunity.microsoft.com/t5/windows-virtual-desktop/public-preview-announcing-public-preview-of-sso-using-ad-fs/m-p/2388588

Configure AD FS single sign-on for Azure Virtual Desktop
https://docs.microsoft.com/ja-jp/azure/virtual-desktop/configure-adfs-sso

RDP Shortpath

リモート デスクトップ クライアントとセッション ホストとの間に直接的な UDP ベースのトランスポートを確立する Azure Virtual Desktop の機能です。

公式アナウンス
https://techcommunity.microsoft.com/t5/windows-virtual-desktop/announcing-public-preview-of-rdp-shortpath-transport-for-windows/m-p/1889372#.X7ILeVWL0as.twitter

Azure Virtual Desktop の RDP Shortpath
https://docs.microsoft.com/ja-jp/azure/virtual-desktop/shortpath

公式アナウンス
https://techcommunity.microsoft.com/t5/windows-virtual-desktop/start-vm-on-connect-starts-public-preview/m-p/2247357

接続時に仮想マシンを起動 (プレビュー)
https://docs.microsoft.com/ja-jp/azure/virtual-desktop/start-virtual-machine-connect
